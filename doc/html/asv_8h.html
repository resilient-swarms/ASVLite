◆ wave_glider_compute_dynamics()

void wave_glider_compute_dynamics ( struct Asv *  asv,
double  rudder_angle,
double  time_step_size 
)

Similar to function asv_compute_dynamics but should be used only for a wave glider. The function sets the position and attitude of the wave glider for the given time step, and also computes the wave thrust force generated by the underwater glider.

Parameters
asvis a non-null pointer to an instance of Asv for which the dynamics is to be computed.
rudder_angleis the angle of the rudder with respect to X axis of the ASV. Rudder angle must within (-PI/2, PI/2). Angle is positive when the vehicle has to turn to right (ie. aft end of the rudder points to starboard side).
time_step_sizein milliseconds to increment the current time.
